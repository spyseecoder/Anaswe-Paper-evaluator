Q1: Solve the quadratic equation 2x² - 8x + 6 = 0
Solution:
2x² - 8x + 6 = 0
Dividing by 2: x² - 4x + 3 = 0
Factoring: (x - 1)(x - 3) = 0
Therefore: x = 1 or x = 3
Verification: 2(1)² - 8(1) + 6 = 2 - 8 + 6 = 0 ✓
             2(3)² - 8(3) + 6 = 18 - 24 + 6 = 0 ✓
Answer: x = 1, x = 3

Q2: Simplify 3x² + 5y - 2x² + 3y - 4
Solution:
Grouping like terms:
= (3x² - 2x²) + (5y + 3y) - 4
= x² + 8y - 4
Answer: x² + 8y - 4
